# Zap Shop React Example

## Installation

1. Install dependencies:
```bash
npm install
```

2. Create a `.env` file in the root directory (optional, for custom configuration):
```env
REACT_APP_SUPRA_RPC_URL=https://rpc-testnet.supra.com
REACT_APP_VERCEL_ENV=development
REACT_APP_STARKEY_WALLET_LINK=https://chromewebstore.google.com/detail/starkey-wallet-qa-test/hcjhpkgbmechpabifbggldplacolbkoh
```

**Note:** In Create React App, all environment variables must start with `REACT_APP_` to be accessible in the browser.

## Development

Run the development server:

```bash
npm start
```

The application will be available at `http://localhost:3000`
